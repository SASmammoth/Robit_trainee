## 과제1 간단한 고찰.

__본 프로그램은 5개의 실수를 입력 받아 최대 최소 평균값을 구하는 코드이다.__

최대/최소 값을 구할 때

double list_max = -10000000000;

double list_min = 10000000000;

위와 같은 형식으로 최댓값 최솟값을 지정해 놓았다. 이때 만약 사용자 입력에서 9999999999999과 같이 최솟값보다 더 큰 값이 5개가 들어오면 최솟값은 9999999999999가 되는 것이 아니라 list_min 인 10000000000 가 된다. 이를 방지하려면 double 범위의 최대 최솟값이 필요한데, 이 예약어는 limits.h 에 있어서 사용할 수 없었다. double의 최대 값인 약±1.8^308이나 64비트를 손으로 입력하는 것도 고민해 보았지만, 딱히 좋은 방법은 아닌 거 같아서 위와 같이 그대로 두기로 하였다. 따라서 입력 값이 정도를 벗어나면 최대 최솟값은 오류가 있을 수 있음. 만약 그러한 입력이 들어오면 사용자에게 알리는 기능을 추가하였다.
