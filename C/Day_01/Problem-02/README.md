## 과제2 간단한 고찰.

_본 프로그램은 정수 N을 입력 받아 N번째 피보나치 수를 출력하는 프로그램이다._

피보나치 수는 이전 항이 2개가 필요한 점화식이라 총 3개의 변수(prev_fibo , now_fibo , next_fibo) 를 사용하여 큐 형식으로 작성하였다. 또한 이전의 피보나치 수 보다 이후의 피보나치 수가 작다면 오버플로우가 일어났다는 소리이니 이를 사용자에게 경고하는 코드도 함께 넣었다. 간단한 플래그 등을 넣어 오버플로우가 발생하면 출력이 안되게 끔 하는 방법도 있지만 경고 만으로도 충분하다고 생각하여 구현하지 않았다.
