# 과제4 간단한 고찰.

_본 프로그램은 5개의 연산자(+-*/^)가 들어갈 수 있는 하나의 식을 입력 받아 계산 결과를 출력하는 프로그램이다._

사칙연산 부분은 딱히 문제가 될게 안 보인다.
하지만 거듭제곱을 계산할 때 pow()함수를 사용할 수가 없어서 정확한 계산이 힘들다. 이를 해결하기 위해 거듭제곱의 우항에 나오는 수를 정수형으로 형변환을 시켜서 반복문에 넣어서 제곱을 계산하였다. 따라서 “2.0 ^ 2.1” 같은 입력은 출력이 4.29가 나오는 게 아니라 4.00과 같이 나오게 된다. 이는 거듭제곱을 간단히 구현하기 위하여 약간의 정확성을 희생한 결과이다. 과제4의 실행 예시에서도 “5.57 ^ 4” 같이 나와 있었기 때문에 거듭제곱의 우측 피연산자는 정수임을 가정하여 과제를 출제하였다고 추측해 볼 수 있다.
입력 형태(%lf %c %lf)와 다른 형태의 입력을 예외 처리 하진 못하지만, ‘!’ 같이 설정하지 않은 연산자가 들어오면 존재하지 않는 연산자라 출력하였다.
